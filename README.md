Проект посвящён тестированию REST API с использованием Postman.
Цель — проверить корректность работы серверной части, включая обработку запросов, валидацию данных и соответствие ответов спецификации.

В рамках проекта протестированы основные CRUD-операции (Create, Read, Update, Delete) и проведён анализ поведения API в позитивных и негативных сценариях.

🎯 Цели тестирования
Проверка корректности работы API-эндпоинтов
Валидация структуры и содержимого ответов (JSON)
Проверка HTTP статус-кодов
Выявление дефектов и нестабильного поведения API
