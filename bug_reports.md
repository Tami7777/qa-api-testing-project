# Bug Reports

## BUG-001: Создание пользователя без данных возвращает успешный статус

**Шаги:**

1. Отправить POST `/users` без body

**Фактический результат:**

* Статус код: 201
* Пользователь создается без данных

**Ожидаемый результат:**

* Статус код: 400
* Сообщение об ошибке

**Серьезность:** Medium
